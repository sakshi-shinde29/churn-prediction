# Customer Churn Prediction using Machine Learning

This project predicts whether a customer will churn (leave) or not using machine learning techniques.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Steps
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training

## Models Used
- Logistic Regression
- Decision Tree

## Result
- Achieved 85% accuracy in predicting customer churn
